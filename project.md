# Growi Plugin開発学習用プロジェクト
- OSSのWikiツールGrowiのプラグイン開発の基礎を学びたいと考えている。
- このプロジェクトは実用性はなくてもよく、もっともシンプルなプラグインの開発に必要な手順を理解するための最低限のファイルや環境設定を行う
- このプラグインをロードするとmarkdown中に$timerと入れると、レンダーエリアやページ閲覧画面にストップウォッチ機能が表示されるようにする
- ストップウォッチの仕様 00:00という表示と、Start/Stopがトグルするボタンと、リセットボタンがある。というもの
